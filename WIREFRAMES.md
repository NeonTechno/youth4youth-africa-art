## Common Design Patterns

### Sections
- **section-padding**: Consistent vertical padding (py-16 md:py-24)
- **container-custom**: Max-width container with horizontal padding
- **bg-neutralLight / bg-white**: Alternating background colors

### Cards
- White or light backgrounds
- Rounded corners (rounded-lg)
- Shadow on hover (hover:shadow-xl)
- Border for definition

### CTAs
- Gradient backgrounds (from-primary to-indigo, etc.)
- Centered text
- Primary and secondary buttons
- Compelling headlines

### Typography Hierarchy
- H1: 4xl md:6xl (hero titles)
- H2: 3xl md:4xl (section titles)
- H3: 2xl md:3xl (subsection titles)
- Body: 16px base, 18px for emphasis

### Color Usage
- Primary: Headings, buttons, accents
- Accent: Highlights, secondary buttons
- Gradients: Hero backgrounds, CTA sections
- Neutral: Body text, backgrounds
